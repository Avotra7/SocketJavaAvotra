Gasql : Mini SGBD socket realise avec le techno java

fonctionalite : -creation base = createdatabase (nom de la base)

				-utilisation base = connect (nom base)

				-base par defaut = data

				-creation table = create (nom de table) (attribut separe par virgule) =>3 mots

				-insertion donnee = insert (nom de table) (donnee separe par virgule) => 3 mots

				-selection de donnee = select (nom de table) => 2 mots

				-selection de tout les tables = show table => 2 mots

				-efface les tables existant = drop (nom de table)

				-efface tout les tables existants = drop all
